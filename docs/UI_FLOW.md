# UI_FLOW.md
> 화면 구조 및 각 페이지 상세 스펙

---

## 1. 전체 화면 구조

```
앱
├── 로그인 화면 (미인증 시 진입점)
└── 메인 레이아웃 (하단 탭 네비게이션)
    ├── 홈 (대시보드)
    ├── 거래 내역
    ├── + 추가 버튼 (FAB)
    ├── 통계
    └── 설정
        ├── 카테고리 관리
        ├── 계좌 관리
        ├── 예산 관리
        └── 데이터 내보내기
```

---

## 2. 네비게이션

### 하단 탭 바 (Bottom Navigation)
```
[홈] [내역] [+추가] [통계] [설정]
```
- 모바일 기준 고정 하단 배치
- `+추가` 버튼은 FAB(Floating Action Button) 스타일로 강조
- 현재 탭 아이콘 색상으로 활성 표시

---

## 3. 페이지별 상세 스펙

### 3-1. 로그인
```
┌─────────────────────┐
│      앱 로고/이름    │
│                     │
│  [이메일 입력]      │
│  [비밀번호 입력]    │
│                     │
│    [로그인 버튼]    │
└─────────────────────┘
```
- 로그인 성공 시 홈으로 이동
- JWT 토큰 localStorage 저장
- 토큰 유효 시 자동 로그인

---

### 3-3. 거래 내역
```
┌─────────────────────┐
│  2024년 1월  [필터▼] │
│                     │
│  1월 15일 화요일    │
│  ├ 🍽 점심식사 -12,000│
│  └ 🚌 버스    -1,400 │
│                     │
│  1월 14일 월요일    │
│  ├ 💰 월급 +3,000,000│
│  └ ☕ 커피   -5,500  │
│  ...                │
│                     │
│  [이번달 합계]      │
│  수입 +500만 / 지출 -320만│
└─────────────────────┘
```
**필터 옵션:** 전체 / 수입만 / 지출만 / 카테고리별
**검색:** 메모, 카테고리명으로 검색

---

### 3-4. 거래 추가/수정
```
┌─────────────────────┐
│  [수입] [지출]       │  ← 탭 전환
│                     │
│     - 12,500원 -    │  ← 금액 (큰 글씨)
│                     │
│  [숫자 키패드]      │
│  1  2  3            │
│  4  5  6            │
│  7  8  9            │
│  00  0  ←           │
│                     │
│  카테고리  [식비 🍽] │
│  계좌     [현금 💵]  │
│  날짜     [2024.01.15]│
│  메모     [입력...]  │
│  정기지출  [  토글 ] │
│                     │
│      [저장하기]     │
└─────────────────────┘
```
- 금액 입력은 커스텀 숫자 키패드 사용 (모바일 UX)
- 카테고리/계좌는 바텀 시트로 선택
- 정기지출 토글 ON 시 → 매월 몇 일 반복 설정

## 4. 공통 UI 컴포넌트

| 컴포넌트 | 용도 |
|---|---|
| BottomSheet | 카테고리/계좌 선택, 필터 |
| AmountInput | 커스텀 숫자 키패드 |
| ProgressBar | 예산 진행률 |
| TransactionItem | 거래 내역 행 |
| MonthPicker | 월 선택 (← YYYY.MM →) |
| CategoryBadge | 카테고리 아이콘 + 색상 |
| EmptyState | 데이터 없을 때 안내 |
